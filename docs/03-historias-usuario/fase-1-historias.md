
## EP-01 – Gestión de usuarios

```md
### HU-01 – Registro de usuario

Como usuario,
quiero registrarme con email y contraseña,
para acceder a la plataforma.

---

### HU-02 – Login de usuario

Como usuario,
quiero iniciar sesión,
para obtener acceso a las funcionalidades del sistema.
```

---

## EP-02 – Autenticación y seguridad

```md
### HU-03 – Generación de token de acceso

Como sistema,
quiero emitir un token al autenticar un usuario,
para validar solicitudes posteriores.

---

### HU-04 – Protección de endpoints

Como sistema,
quiero restringir endpoints según autenticación,
para evitar accesos no autorizados.
```

---

## EP-03 – Roles y permisos

```md
### HU-05 – Creación de roles

Como administrador,
quiero crear roles,
para definir distintos niveles de acceso.

---

### HU-06 – Asignación de roles a usuarios

Como administrador,
quiero asignar roles a usuarios,
para controlar permisos.
```

---

## EP-04 – Infraestructura y persistencia

```md
### HU-07 – Persistencia de usuarios

Como sistema,
quiero guardar los datos de los usuarios,
para mantener información consistente.

---

### HU-08 – Migraciones de base de datos

Como desarrollador,
quiero manejar migraciones,
para versionar cambios en la base de datos.
```

## EP-05 – Arquitectura base

```md
### HU-09 – Estructura del proyecto

Como desarrollador,
quiero una estructura de carpetas clara,
para mantener el código ordenado.

---

### HU-10 – Separación por capas

Como desarrollador,
```
